En este ejercicio se nos pide mostrar los datos de un empleado que trabaja en un Departamento concreto solicitado por el usuario.
Para ello pediremos el ID_departamento y realizaremos una consulta con un where: "WHERE DEPARTAMENTO_ID = ?" para que a la hora de mostrar los datos con un ResultSet se nos muestren solo los empleados que trabajen en dicho departamento
